# User Knowledge Files

This directory contains combined knowledge files generated when a mailbox item is closed.

Each file merges Legal Beagle's topic overview with your specific preferences, answers, and decisions — giving you a personalized reference for that topic.

## File naming

`LB-NNN-<slug>.md` — matches the mailbox item ID.

## File structure

Each file contains:
- **Topic summary** — what the knowledge file covers
- **Your answers / preferences** — what you told Legal Beagle
- **Tailored next steps** — actions relevant to your situation
- **Date captured**
